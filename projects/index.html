Other Personal Projects

I do a lot of personal projects. Somethimes I even write about them! I'm trying to do that more. Here's some of the write-ups I've done so far

You should check this page that lists my github projects instead, it has many more projects! This page has some write-ups you won't find over there though.


RIT COVID Dashboard https://ritcoviddashboard.com

My school, the Rochester Institute of Technology, provides no way to see historical data. I wrote a scraper in Python that saves data to a SQLite database, expose all the data via a JSON API, and then display it as a React app. You can read more about the API and dashboard here.



FastBin

FastBin is a quick project I wrote that provides client-side encryption for text snippets. A core idea is for the encryption secret to never be sent to the server. You can read more about FastBin here.



Spotify Recent Plays

A small project to display the songs I've recently listened to on Spotify. It draws directly from the spotify API using an ASP.NET app and displays it using a TypeScript/React app. You can read more about the Spotify Recent Plays app here.
